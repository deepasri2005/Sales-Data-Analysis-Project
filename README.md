# Sales-Data-Analysis-Project
"Data Cleaning, Analysis, and Dashboard Project using Raw Sales Data"

## Overview
This project demonstrates **data cleaning, analysis, and visualization** using raw sales data. It is designed for aspiring data analysts to showcase practical skills in Python, SQL, and Power BI.
---
## Dataset
- **Raw Dataset**: `raw_sales_dataset.csv` (2500+ rows, 14 columns)  
- **Cleaned Dataset**: `sales_cleaned.csv`  
- Dataset contains missing values, duplicates, and mixed formats for practice in **data cleaning**.
---
## Tools & Technologies
- Python (pandas, matplotlib)  
- SQL  
- Microsoft Power BI  
- Git & GitHub  
---
## Steps Performed

1. **Data Cleaning**
   - Removed duplicates
   - Handled missing values
   - Converted date columns
   - Ensured numeric columns

2. **Data Analysis**
   - Total Sales & Profit
   - Sales by Region
   - Profit by Category
   - Top 5 Products by Sales
   - Average Discount

3. **Visualization**
   - Bar chart: Sales by Category
   - Pie chart: Sales by Region
   - Line chart: Profit over time

4. **SQL Queries**
   - Summarized sales and profit
   - Grouped analysis by region, category, and products

5. **Power BI Dashboard**
   - Created visuals for business insights
   - Interactive filters and KPIs for decision-making

## Key Insights
- Highest sales region: West  
- Most profitable category: Technology  
- Some products have high sales but low profit due to discounts  
- Monthly profit trends indicate peak months for business planning  

---

## How to Run

1. Open `analysis.ipynb` in Google Colab or Jupyter Notebook  
2. Run cells step by step (data cleaning → analysis → visualization)  
3. Optional: Run SQL queries in SQLite  
4. Open `dashboard.pbix` in Power BI for interactive visualization
